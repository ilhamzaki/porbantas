# Porbantas - Portal Berita Tanpa Batas

## Deskripsi

Porbantas merupakan sebuah website yang menampilkan berita-berita dari sumber yang terpercaya dan ter-update dengan langsung membaca dari sumbernya

## Demo

[https://ilhamzaki.github.io/porbantas.github.io/](https://ilhamzaki.github.io/porbantas.github.io/?#/)

## API yang digunakan

- [API Berita Indonesia](https://github.com/renomureza/api-berita-indonesia)
- [API Covid-19 Indonesia](https://github.com/Reynadi531/api-covid19-indonesia-v2)
- [API Vaksinasi Indonesia](https://cekdiri.id/vaksinasi/)

## Instalasi

1. Clone: `git clone https://github.com/ilhamzaki/porbantas.git`
2. Install dependecies: `npm i`
3. Run server: `npm run start-dev`
